A python discord bot for my private sever

## Installation

Clone the Repo and install requirements:

```bash
pip install -r requirements.txt
```

Create a config.py file and put:

```bash
token = 'YOURDISCORDTOKEN'
api = 'GENIUSAPI'
```


## Usage


Run main.py
